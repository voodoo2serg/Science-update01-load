# Science Concierge — hardened server-ready branch

Это усиленная версия MVP для развёртывания на Debian 13 одной командой.

## Что улучшено
- убраны критические проблемы форматирования Python-кода;
- добавлена базовая policy-защита для LLM;
- добавлена проверка типа и размера загружаемых файлов;
- Nginx вынесен перед backend;
- n8n закрыт базовой аутентификацией;
- backend и bot работают не от root;
- добавлены healthchecks и более безопасный compose;
- есть `scripts/install.sh` для запуска одной командой.

## Установка на Debian 13
1. Установить Docker Engine и Docker Compose plugin.
2. Скопировать репозиторий на сервер.
3. Выполнить:

```bash
bash scripts/install.sh
```

Перед первым реальным запуском обязательно заполните `BOT_TOKEN` в `.env`.

## Дисклеймер
Сервис не пишет готовые научные работы и не помогает обходить антиплагиат. Он дает только безопасные черновые каркасы и методические рекомендации.

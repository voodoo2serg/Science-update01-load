# Science Concierge MVP

MVP-система для сервиса сопровождения соискателей ученых степеней.

## Что входит
- FastAPI backend
- PostgreSQL
- Telegram bot (aiogram)
- n8n для автоматизации
- Docker Compose для локального и серверного деплоя

## Функции MVP
- регистрация пользователя из Telegram
- хранение профиля и статуса подписки
- загрузка файлов
- генерация безопасного черновика плана статьи или главы
- проверка доступа по подписке
- запрос на экспертизу
- сбор обратной связи по ответам
- healthcheck и базовые тесты

## Быстрый старт
1. Скопируйте `.env.example` в `.env`
2. Заполните `BOT_TOKEN`
3. Запустите:

```bash
docker compose up --build -d
```

Backend будет доступен на `http://localhost:8000/docs`

## Структура
- `backend/` — API и бизнес-логика
- `bot/` — Telegram bot
- `deploy/` — конфиги деплоя

## Production
- закройте доступ к порту PostgreSQL извне
- используйте reverse proxy и HTTPS
- вынесите `.env` в защищенное хранилище секретов
- настройте резервное копирование PostgreSQL
- при необходимости подключите S3/MinIO вместо локального `STORAGE_DIR`

## Важный дисклеймер
Система не генерирует готовые диссертации или статьи “под ключ”. Она выдает структуру, тезисы, проверочные рекомендации и черновые материалы, требующие самостоятельной переработки автором.
